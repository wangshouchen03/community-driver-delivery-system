# 🚚 社区司机物流配送接单系统

基于纯 HTML/CSS/JavaScript 构建的单页面物流配送接单管理系统，无需任何后端依赖，打开即用。

## ✨ 功能特性

- 📋 **订单管理** — 浏览、接单、配送追踪、状态管理
- 💬 **即时通讯** — 司机与客户内置聊天功能
- 📊 **收入统计** — 历史收入记录与数据分析
- 👤 **司机档案** — 个人信息、车辆信息、评价管理
- 🔐 **登录注册** — 独立注册页面，手机号验证
- 📱 **响应式设计** — 适配手机端，微信可直接打开
